# 🌈 单词小勇士 - Word Hero

一款面向 3-6 岁儿童的互动式单词学习游戏，通过身体追踪技术让孩子用身体动作选择答案！

## ✨ 功能特点

### 🎮 游戏玩法
- **身体追踪选择**：使用 MediaPipe Pose 检测用户位置（左/中/右）
- **单词-图片匹配**：屏幕显示单词，孩子站到对应图片的位置选择答案
- **渐进式学习**：可自由选择要学习的单词

### 🏆 计分系统
- **基础得分**：每答对一题得 100 分
- **连击加成**：连续答对，倍率最高 5x
- **浮动分数**：答对时显示 "+XX" 动画

### 🎨 视觉效果
- **可爱童趣 UI**：糖果色系、圆角设计
- **骨骼发光效果**：检测到的身体骨骼带有发光特效
- **庆祝特效**：答对时彩纸飞舞动画
- **镜像模式**：摄像头画面水平翻转

### 🎵 音乐功能
- 支持上传本地音乐文件
- 音频可视化显示

### 🤖 AI 集成
- 集成 Google Gemini AI
- 根据得分生成动态鼓励语
- 支持用户输入 API Key 激活

## 🚀 快速开始

### 启动服务器
```bash
cd ~/CascadeProjects/word-picture-game
node server.js
```

或使用启动脚本：
```bash
chmod +x start.sh
./start.sh
```

### 访问游戏
打开浏览器访问：http://localhost:3000

### 停止服务器
```bash
./stop.sh
```

## 📱 使用说明

1. **加载页面**：等待游戏资源加载完成
2. **选择单词**：点击要学习的单词，然后点击"开始游戏"
3. **游戏界面**：
   - 允许摄像头权限
   - 屏幕上方显示当前单词
   - 下方显示三张图片（左/中/右）
   - 站到对应位置 1.5 秒选择答案
4. **得分**：答对得分，连续答对获得连击加成

## 🛠️ 技术栈

- **React 18** (CDN)
- **Tailwind CSS**
- **MediaPipe Pose** - 身体追踪
- **Web Audio API** - 音频分析
- **Google Generative AI** - Gemini AI 集成

## 📋 题库

包含 16 个适合幼儿的基础单词：
- 水果：Apple, Banana
- 动物：Cat, Dog, Fish, Bird, Rabbit, Elephant
- 自然：Sun, Moon, Star, Flower, Tree
- 物品：Ball, Car, House, Cake

## ⚠️ 注意事项

- 需要摄像头权限
- 建议在光线充足的环境使用
- 确保摄像头能看到完整身体
- AI 功能需要有效的 Gemini API Key
